# 🧠 Memory Game - 숫자 기억력 게임

> 파이게임을 활용한 **숫자 기억력 테스트 게임**입니다.  
> 숫자들이 잠깐 화면에 표시된 뒤 사라지고, 사용자는 **1부터 순서대로** 버튼을 클릭하여 기억력을 테스트하게 됩니다.

---

## 📌 기능 소개

- 난이도(레벨)에 따라 숫자의 개수와 보여주는 시간이 자동으로 조절됩니다.
- 사용자가 **올바른 순서로 숫자를 클릭**하면 다음 레벨로 진행됩니다.
- **틀린 숫자를 클릭**하면 게임 오버와 함께 최종 레벨을 출력합니다.
- 레벨이 올라갈수록 숫자 수가 늘어나고, 표시 시간은 짧아집니다.

---

## 🎮 게임 방법

1. 실행 후 나오는 **동그란 버튼**을 클릭하면 게임이 시작됩니다.
2. 화면에 숫자가 표시되며, 일정 시간이 지나면 숫자가 사라집니다.
3. 숫자가 **사라진 위치를 기억해 1부터 순서대로 클릭**하세요.
4. 정답이면 다음 레벨로 진행, 오답이면 게임이 종료됩니다.

---

## 🧠 레벨 시스템

- `display_time = 5 - (level // 3)` 으로 난이도 상승
- 숫자 개수는 `(level // 3) + 5` (최대 20개)
- 초반에는 여유롭게, 레벨이 오를수록 기억력이 많이 필요해집니다!
